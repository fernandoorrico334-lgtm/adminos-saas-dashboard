# AdminOS | Painel Administrativo SaaS

Site/painel administrativo escuro e profissional para portfólio, criado em HTML e CSS puro. O projeto apresenta uma experiência vertical com quatro áreas reais de um SaaS: visão executiva, financeiro/CRM, central de suporte e operação enterprise.

## Áreas do painel

- **Visão Executiva:** KPIs, receita recorrente, composição de planos, top clientes e alertas.
- **Financeiro & CRM:** faturamento, recebimentos, inadimplência, faturas e pipeline comercial.
- **Central de Suporte:** chamados, SLA, filas de atendimento, atividade do time e mapa mundi de saúde dos clientes.
- **Operação Enterprise:** governança, indicadores, clientes, configurações rápidas e rotinas administrativas.

## Recursos

- Aparência dark mode profissional.
- Layout vertical, pronto para ser usado como portfolio.
- Componentes densos de painel administrativo: cards, tabelas, gráficos, navegação, status e mapa.
- Mapa mundi em SVG com rotas e marcadores de saúde.
- Totalmente responsivo.
- Sem dependências externas.

## Stack

- HTML5
- CSS3

## Como rodar

Abra o arquivo `index.html` no navegador.

## Objetivo de portfólio

Demonstrar capacidade de construir interfaces SaaS modernas, com linguagem visual de produto real, fluxo corporativo e composição profissional para apresentação a clientes.
